💡 Satz 1: AB = Stück ZWISCHEN den Parallelen  |  Satz 2: SB = ganzer Weg von S aus
📌 1. Strahlensatz — SA/AB = SC/CD
SA / AB = SC / CD
AB liegt ZWISCHEN den Parallelen (nicht von S aus!)
Schieben: 38%
Tafel-Beispiel (Golden Gate)

· 2,5h = (601/5) · 2,5
=h = 300,5 m

📌 2. Strahlensatz — SA/SB = SC/SD
SA / SB = SC / SD
SB = ganzer Weg von S bis zur zweiten Parallele
🧠 Merktrick: S→A→B = SB. "B" beginnt bei S, nicht bei A!
Schieben: 38%
Übung
SA = 4, SB = 12, SC = 3 → SD = ?
SD =
4/12 = 3/SD → SD = 3 · 12/4 = 9
📌 3. Strahlensatz — SA/SB = AC/BD
SA / SB = AC / BD
Die Parallelen wachsen proportional
Je weiter von S → desto länger die Parallele. Doppelter Abstand = doppelt so lang!
SA = 3, SB = 9, AC = 4 → BD = ?
BD =
3/9 = 4/BD → BD = 4 · 9/3 = 12

📌 Alle Formeln — wie auf dem Projektor

A = π · r²

KREISRING

KREISSEKTOR

b = Kreisbogenabschnitt  |  ω = Mittelpunktswinkel  |  δ = Umfangswinkel
