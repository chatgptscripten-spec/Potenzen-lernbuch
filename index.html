📌 Strahlensätze — Alle 3 Formeln
1. Strahlensatz
SA/AB = SC/CD
AB = zwischen den Parallelen
2. Strahlensatz
SA/SB = SC/SD
SB = ganzer Weg von S aus
3. Strahlensatz
SA/SB = AC/BD
Parallelen wachsen proportional zum Abstand
⭕ Kreis-Formeln
Umfang
u = 2·π·r
oder u = π·d
Fläche
A = π·r²
r = Radius
Bogenlänge
b = 2πr·ω/360°
ω = Mittelpunktswinkel
Kreissektor
A = πr²·ω/360°
auch: A = b·r/2
Kreisring
A = π·(ra²−ri²)
ra = außen, ri = innen
Rechteck-Umfang
u = 2a + 2b
a, b = Seiten
🔵 Satz des Thales
Wenn ω = 180° (Mittelpunktswinkel über dem Durchmesser), dann ist der Umfangswinkel δ = 90°.

Merke: Jeder Winkel im Halbkreis ist ein rechter Winkel!
🚨 Häufige Fehler
❌ Satz 1 vs 2 verwechseln → Satz 1: AB zwischen · Satz 2: SB von S aus
❌ Kreisring: nur eine Fläche → Immer beide r² berechnen, dann subtrahieren!
❌ Sektor: falscher Winkel → ω in Grad, immer mit ω/360°
❌ Thales vergessen → ω = 180° → δ = 90° immer!
