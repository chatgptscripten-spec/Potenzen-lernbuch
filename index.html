Potenzen · Kapitel 1

Was ist eine Potenz?

Kurze Schreibweise für wiederholte Multiplikation

🎯
📖 Erklärung:
an = a · a · … · a
Basis a wird n-mal mit sich selbst multipliziert · a⁰ = 1 immer
📌
Basis = große Zahl unten  |  Exponent = kleine Zahl oben  |  a⁰ = 1 für alle a ≠ 0
💡 Die Idee

Statt 3 · 3 · 3 · 3 schreiben wir einfach 3⁴. Der Exponent sagt, wie oft die Basis multipliziert wird.

an = a · a · a · … · a
„a hoch n" — n Faktoren
📌
Basis (a) = die Zahl die multipliziert wird
Exponent (n) = wie oft sie multipliziert wird
Potenzwert = das Ergebnis
📝 Beispiel: 3⁴ ausrechnen
Aufgabe3⁴ = „drei hoch vier"
Ausschreiben= 3 · 3 · 3 · 34 Faktoren
Schritt 1= 9 · 9 = 81
3⁴ = 81
⭐ Sonderfälle — auswendig lernen!

⬛

a⁰ = 1 gilt für alle Zahlen außer 0 — auch 1000⁰ = 1 und 0,5⁰ = 1!

Potenzen · Kapitel 2

Vorzeichen-Regeln

Was passiert wenn die Basis negativ ist?

➕
📖 Erklärung:
GERADE Exp. → ➕
(−3)² = +9
(−2)⁴ = +16
UNGERADE Exp. → ➖
(−3)³ = −27
(−2)⁵ = −32
🚨
−3² = −9 ≠ (−3)² = +9  →  Klammer macht den Unterschied!
🧠 Warum gerade/ungerade?

Wenn du (−3)⁴ ausschreibst: (−3)·(−3)·(−3)·(−3). Je zwei Minus heben sich auf — bei einer geraden Anzahl bleibt kein Minus übrig. Bei ungerader Anzahl bleibt ein Minus.

GERADER Exponent → POSITIV
(−3)² = (−3)·(−3) = +9 ✅
(−2)⁴ = +16 ✅
(−5)⁶ = +15625 ✅
Alle Minus heben sich auf
UNGERADER Exponent → NEGATIV
(−3)³ = −27 ✅
(−2)⁵ = −32 ✅
(−4)¹ = −4 ✅
Ein Minus bleibt übrig
🚨 Die Klammer-Falle
🚨
−3² = erst 3²=9 rechnen, dann − davor → = −9
(−3)² = Minus gehört zur Basis, gerade → = +9

Ohne Klammer: das Minus steht außerhalb und wird nicht potenziert!

Potenzen · Kapitel 3

Zehnerpotenzen

Den Exponenten kennen = die Nullen kennen!

🔟
📖 Erklärung:
10n = 1 mit n Nullen
10⁵ = 100.000  |  10⁻² = 0,01
🔟 Zehnerpotenzen verstehen

Bei Zehnerpotenzen ist die Basis immer 10. Der Exponent sagt direkt, wie viele Nullen die Zahl hat.

10n = 1 gefolgt von n Nullen
Exponent = Anzahl der Nullen
PotenzWertName
10¹10Zehn
10²100Hundert
10³1.000Tausend
10⁴10.000Zehntausend
10⁶1.000.000Million
💡
Zehnerpotenzen überall: Megabyte = 10⁶ Byte, ein Kilometer = 10³ Meter!

Potenzen · Kapitel 4

Potenzen multiplizieren

Gleiche Basis → Exponenten addieren

✖️
📖 Erklärung:
an · am = an+m
Gleiche Basis → Exponenten ADDIEREN  |  5³·5⁴ = 5⁷
🚨
5³ · 5⁴ = 5⁷ ✅  NICHT  5¹² ❌ — Exponenten addieren, nicht multiplizieren!
🧮 Warum addieren wir?

Wenn du 5³ · 5⁴ ausschreibst, siehst du 3 + 4 = 7 Fünfen:

an · am = an+m
Gleiche Basis → Exponenten ADDIEREN
Aufgabe5³ · 5⁴
Ausschreiben(5·5·5) · (5·5·5·5) = sieben 5en
Kurzform= 5⁷3+4=7
5⁷ = 78.125
🚨
5³ · 5⁴ = 512 ❌ → Die Exponenten werden NICHT multipliziert!
5³ · 5⁴ = 57 ✅ → Sie werden ADDIERT!

Potenzen · Kapitel 5

Potenzen dividieren

Gleiche Basis → Exponenten subtrahieren

➗
📖 Erklärung:
an : am = an−m
Gleiche Basis → Exponenten SUBTRAHIEREN  |  6⁷:6³ = 6⁴
💡
Sonderfall: an : an = a⁰ = 1
🧮 Division durch Kürzen

Wenn du 6⁵ durch 6² teilst, kannst du oben und unten je zwei 6en wegkürzen. Es bleiben 3 Sechsen übrig → 6³.

an : am = an−m
Gleiche Basis → Exponenten SUBTRAHIEREN
Aufgabe6⁵ : 6²
Kürzen(6·6·6·6·6) ÷ (6·6) = 6·6·6
Ergebnis= 6³5−2=3
6³ = 216
💡
Sonderfall: 7⁴ : 7⁴ = 7⁰ = 1  (denn 4−4=0, und a⁰ = 1!)

Potenzen · Kapitel 6

Potenz einer Potenz

Klammer → Exponenten multiplizieren!

🔄
📖 Erklärung:
(an)m = an·m
Klammer → Exponenten MULTIPLIZIEREN  |  (3²)⁵ = 3¹⁰
⚠️
Klammer = ×  |  Kein Klammer = +  →  (4³)² = 4⁶ aber 4³·4² = 4⁵
🔄 Potenz der Potenz

(4³)² bedeutet: 4³ wird zweimal multipliziert → 4³ · 4³ = 4⁶. Oder direkt: 3 × 2 = 6.

(an)m = an·m
MIT Klammer → Exponenten MULTIPLIZIEREN
Aufgabe(4³)²
Bedeutet4³ · 4³ (zweimal)
Kurz= 4^(3·2) = 4⁶3×2=6
4⁶ = 4096
⚠️
4³ · 4² = 4⁵  (ohne Klammer → ADDIEREN: 3+2=5)
(4³)² = 4⁶  (MIT Klammer → MULTIPLIZIEREN: 3×2=6)

Potenzen · Kapitel 7

Brüche potenzieren

Zähler UND Nenner bekommen den Exponenten

🍕
📖 Erklärung:
(ab)n = anbn
Zähler und Nenner einzeln potenzieren
💡
(²⁄₃)² = 4/9  |  (½)³ = 1/8  |  Vorzeichen-Regel gilt wie immer!
🍕 Brüche hochrechnen

Bei einem Bruch hoch n wird der Exponent auf Zähler UND Nenner getrennt angewendet.

(ab)n = anbn
Zähler UND Nenner einzeln potenzieren!
Aufgabe(²⁄₅)³
Zähler2³ = 8
Nenner5³ = 125
8/125
💡
Vorzeichen gelten wie immer: (−²⁄₃)² = +4/9 (gerade → positiv), (−²⁄₃)³ = −8/27 (ungerade → negativ)

📦 Produkt potenzieren
(a · b)n = an · bn
Potenz auf alle Faktoren verteilen!
Aufgabe(3 · 4)²
Verteilen= 3² · 4²
Rechnen= 9 · 16 = 144
144 (= 12² — denn 3·4=12, 12²=144 ✓)

Negative Exponenten

Minus im Exponent = Kehrwert bilden!

➖

a−n = 1an

➖ Was bedeutet negativer Exponent?

Ein negativer Exponent dreht die Potenz um den Kehrwert. Das Minus bedeutet: „1 geteilt durch".

a−n = 1an
Minus → Kehrwert der Potenz

Kehrwert= 1 / 4² = 1/16

Von 0 (unmöglich) bis 1 (sicher)

🎲

P(A) = günstige Fällealle möglichen Fälle

P(A) = günstige Fällealle möglichen Fälle

Wahrscheinlichkeit · Kapitel 3

Aus einer Urne ziehen

Kugeln, Karten, Lose — zählen & berechnen

🔵
📖 Erklärung:
💡
1. Gesamtanzahl der Objekte bestimmen (= Nenner)
2. Objekte die das Ereignis erfüllen zählen (= Zähler)
3. P = Zähler/Nenner  |  ×100 für Prozent
🔵 Wie funktioniert das Ziehen?

Du hast eine Urne (Schachtel, Sack, Glas) mit verschiedenen Objekten. Du greifst blind rein und ziehst ein Objekt. Welche Chance hast du, etwas Bestimmtes zu ziehen?

Schritt 1Alle Objekte zählen → das ist der Nenner
Schritt 2Günstige Objekte zählen → das ist der Zähler
Schritt 3P = Zähler ÷ Nenner
💡
Urne mit 5 roten, 3 blauen, 2 grünen Kugeln (10 gesamt):
P(rot) = 5/10 = 1/2 = 50%
P(blau) = 3/10 = 30%
P(nicht grün) = 8/10 = 4/5 = 80%
📊 Erwartungswert bei mehreren Versuchen
E = P · n
Erwartete Treffer = Wahrscheinlichkeit × Anzahl Versuche
💡
P(rot) = 1/2, 40 Versuche → E = 1/2 · 40 = 20 rote Kugeln erwartet
P(blau) = 3/10, 50 Versuche → E = 3/10 · 50 = 15 blaue erwartet

Häufigkeit · Kapitel 1

Absolute & relative Häufigkeit

Zählen, vergleichen, in Relation setzen

📊
📖 Erklärung:
📌
Absolute Häufigkeit = das Zählergebnis direkt (z.B. 14 Schüler)
Relative Häufigkeit = abs. H ÷ Gesamtzahl N  →  ×100 für %
Summe aller rel. Häufigkeiten = immer genau 1 (= 100%)
rel. H. = absolute HäufigkeitGesamtzahl N
📖 Die zwei Arten
Absolute Häufigkeit
= das Zählergebnis selbst
Wie oft kommt X vor?
Einheit: Anzahl (Stück)
„Das eigentliche Zählergebnis"
Relative Häufigkeit
= Anteil an Gesamtzahl
abs. H. ÷ N
Als Bruch, Dezimal oder %
„Der Anteil von der Gesamtmenge"
rel. H. = abs. H.N
N = Gesamtanzahl aller Werte  |  ×100 für Prozent
📝 Beispiel: Lieblingsfarbe in Klasse 8b (30 Schüler)
📋
Umfrageergebnis: Blau=12, Rot=7, Grün=5, Gelb=4, Andere=2  |  Gesamt: 30
Farbeabs. H.BruchDezimal%
Blau1212/30 = 2/50,440%
Rot77/30≈0,233≈23,3%
Grün55/30 = 1/6≈0,167≈16,7%
Gelb44/30 = 2/15≈0,133≈13,3%
Andere22/30 = 1/15≈0,067≈6,7%
GESAMT3011,0100%
📝 Schritt-für-Schritt Vorgehen
1Alle absoluten Häufigkeiten aufschreiben
2Addieren → Gesamtzahl N
3rel. H. (Bruch) = abs. H. ÷ N  (kürzen!)
4Dezimal = Bruch ausrechnen (÷)
5Prozent = Dezimal × 100
⚠️
Probe: Alle rel. Häufigkeiten addieren → muss genau 1 (= 100%) ergeben!
